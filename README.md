# DesklyCRM 🚀

Современная и легкая CRM-система для малого бизнеса. Полностью переписана на **Go** и **React** в марте 2026 года для максимальной производительности.

## Стек технологий
- **Backend**: Go 1.22+ (GORM, SQLite, JWT)
- **Frontend**: React + TypeScript + Vite (Lucide-React, Glassmorphism UI)
- **Database**: SQLite (локальный файл `deskly.db`)

## Основные возможности
- **Дашборд**: Реальная статистика оборота, выручки и количества клиентов в реальном времени.
- **Заказы**: Умное управление статусами, авто-расчет стоимости на основе товаров, каскадное удаление.
- **Финансы**: Система учета платежей (возможность оплаты заказов частями).
- **Клиенты**: База с быстрым поиском и историей заказов.
- **Товары**: Гибкий каталог услуг с возможностью архивации.

## Как запустить (Backend + Frontend вместе)

1. Убедитесь, что установлен **Go 1.22+**.
2. Перейдите в папку бэкенда:
   ```bash
   cd go-backend
   ```
3. Запустите сервер (он сам подтянет скомпилированный фронтенд):
   ```bash
   go run ./cmd/server/main.go
   ```
4. Откройте в браузере: **http://localhost:8080**

## Данные для входа (дефолтные)
- **Email**: `admin@deskly.com`
- **Password**: `password`

## Разработка фронтенда
Если вы хотите внести изменения в интерфейс:
1. Перейдите в `go-backend/frontend`.
2. Запустите `npm install` и `npm run dev`.
3. После правок соберите билд: `npm run build`.
